# 多平台视频链接解析与下载引擎

一个合规、可扩展、可维护的视频链接解析与下载系统。

## 功能特性

- 自动识别视频平台类型
- 解析视频元数据（标题、封面、分辨率、时长、编码）
- 在平台允许的前提下返回可下载的视频资源
- 优先返回无水印或官方高清源
- 不可下载时明确告知原因并给出替代方案

## 支持的平台

### 国内平台
- 哔哩哔哩（Bilibili）
- 抖音（Douyin）
- 快手
- 小红书
- 腾讯视频
- 爱奇艺

### 国际平台
- X / Twitter
- YouTube（仅限公开视频、非 DRM）
- Vimeo
- Dailymotion

## 合规性

本工具严格遵守以下原则：
- 仅解析平台允许下载的公开视频内容
- 不支持付费、会员或受 DRM 保护的视频
- 尊重原平台版权与使用协议
- 所有下载行为需明确提示："仅限个人学习与平台允许范围内使用"

## 部署方式

本项目提供三种部署方式：
1. 网站手动部署版
2. 前后端分离版
3. Docker快速部署版

详细文档请参见 [docs](./docs) 目录。

## 项目结构


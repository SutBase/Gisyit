## 프로젝트 개요

이 프로젝트는 **사용자 주도 제품 실험**으로, 플랫폼의 방향은 커뮤니티에 의해 결정됩니다.

이 프로젝트는 **“아이디어 크라우드소싱 + 내가 구현”** 모델을 따릅니다. 사용자가 실제 요구사항과 아이디어를 제안하면, 저는 시스템 아키텍처 설계, 엔지니어링 구현, 지속적인 반복 개선을 담당합니다. 기능 우선순위는 **커뮤니티 피드백과 투표**에 의해 결정되며, 추측이나 폐쇄형 결정은 없습니다.

---

## 배경

이 프로젝트는 상업적 계획에서 시작된 것이 아니라, 개인적인 흥미와 실험에서 시작되었습니다.  
**투명하고 커뮤니티 주도적인 제품 개발 방식을 검증**하기 위해 설계된 유연한 환경입니다.

- 엄격한 마감일이나 KPI 없음  
- 품질, 유지보수 가능성, 실현 가능성을 속도보다 우선시

---

## 현재 진행 상황

- **400+ 아이디어와 제안** 수집 (국내외 사용자)  
- **12개의 실사용 애플리케이션** 구현  
- 다수의 **무료 AI 도구** 제공  
- 지속적인 업데이트와 확장 진행 중  

모든 기능은 **사용 가능하고, 유지 가능하며, 재사용 가능한 최소 기준**을 충족합니다.

---

## 기존 애플리케이션

현재 개발 중이거나 사용 가능한 도구:

- **AI PPT**: 구조화된, 바로 사용 가능한 프레젠테이션 콘텐츠 자동 생성   
- **AI Table**: 복잡한 데이터를 읽기 쉽고 편집 가능한 표로 변환  
- **CloudNote**: 장기 사용을 위한 클라우드 기반 구조화 노트 시스템  
- **AI Studio**: 다중 모델, 다중 시나리오 AI 실험 및 창작 공간  

이 도구들은 **생산성 중심**이며, 단순한 엔터테인먼트 도구가 아닙니다.

## 공동 창작 원칙

### 1. 아이디어 평가
모든 아이디어는 검토 및 평가됩니다. 선택 기준:

- 기술적 실현 가능성  
- 확장성 및 장기 유지 가능성  
- 실제 사용 가치  

인기나 트렌드는 구현 결정에 영향을 미치지 않습니다. **엔지니어링 중심 프로젝트**입니다.

### 2. 속도 및 기대
- 명확한 일정이나 속도 보장 없음  
- 품질, 실현 가능성, 장기 유지보수에 집중

### 3. 존중과 평등
- 모든 아이디어를 존중, 짧은 제안이라도 평가  
- 기여는 코드 라인 수가 아니라 아이디어의 가치로 측정

## 오픈소스 & 소유권

이 프로젝트 이전에, 사용자 피드백 기반으로 여러 애플리케이션을 이미 오픈소스화했습니다.  
이번 프로젝트 종료 후:

- 각 기여자에게 애플리케이션을 오픈소스로 공개할지 여부를 문의  
- 오픈소스화는 **완전히 자율적**, 강제 없음  

**중요:** 채택된 모든 아이디어에 대해서는 해당 애플리케이션의 **전체 소스 코드가 기여자에게 제공**됩니다. 소스 코드, 제어권, 최종 결정권이 기여자에게 있습니다.

## 참여 대상

다음과 같은 경우 참여를 권장합니다:

- 실제 요구나 아이디어는 있으나 엔지니어링 자원이 없는 경우  
- **투명하고 검증 가능한 공동 창작 실험**에 기여하고 싶은 경우  

명확한 일정, KPI 또는 상업적 보장을 기대한다면, 이 프로젝트는 적합하지 않을 수 있습니다.

## 프로젝트 철학

- 산업을 바꾸겠다는 약속 없음  
- 빠른 성공을 보장하지 않음  

하지만 채택된 모든 아이디어는 **진지하게 고려**되며, **실제 코드**로 존재합니다.  

이 프로젝트는 **먼저 구축하고, 나중에 평가하는 방식**입니다. 이 접근 방식에 공감한다면 참여를 환영합니다.
